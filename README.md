# Git Commands:

- `git add`

- `git commit`

- `git push`

- `git clone`
  This just downloads files

- `git fetch`
  Check whether remote changed
  
- `git pull`
  Pulls changed from remote

- `git status`
  This says what changed
  
- `git reset --hard`
  You messed up your code and you want to go back (fuk go back button)
